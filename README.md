•	📁 Interactive Digital Portfolio Website

An interactive, responsive digital portfolio website built using HTML, CSS, and JavaScript.
This portfolio showcases projects, skills, resume, and contact details with dynamic UI elements
and smooth animations to enhance user experience

🚀 Features
Responsive Design – Fully functional on desktop, tablet, and mobile.

Smooth Scroll Navigation – Easily navigate between sections.

Animated Sections – Elements fade in or animate on scroll.

Skills Section – Visual representation of technical skills.

Contact Form – Validated contact form (JavaScript).




🛠️ Built With

HTML5 – Semantic structure and accessibility.

CSS3 – Responsive layout using Flexbox/Grid, animations, and transitions.

JavaScript (Vanilla) – DOM manipulation, interactivity, event handling.

🔧 How to Use

Open in browser:
Open index.html in your preferred browser.

Customize:

Update personal information, images, and links in index.html.
Add your own projects and descriptions in the Projects section.
Modify styles in css/style.css to match your branding.
Add more interactive features in js/script.js.
